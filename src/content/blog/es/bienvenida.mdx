Después de 15 años desarrollando software, he decidido abrir este espacio para
compartir lo que voy aprendiendo. Aquí escribiré sobre **PHP**, arquitectura de
aplicaciones, buenas prácticas y todo lo que me parezca útil.

## ¿Por qué un blog sin base de datos?

Este blog está construido con [Astro](https://astro.build) y Markdown/MDX. Cada
artículo es un fichero, lo que significa:

- **Cero base de datos** que mantener.
- **Máxima velocidad**: todo se genera como HTML estático.
- **Control de versiones** con Git para cada publicación.
